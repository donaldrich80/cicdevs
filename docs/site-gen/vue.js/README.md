## Develop in Docker

### Command

```sh
docker pull donaldrich/vue.js:latest
docker run -it --rm \
--hostname=vue.js \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/zsh" \
--net="host" \
donaldrich/vue.js:latest
```
