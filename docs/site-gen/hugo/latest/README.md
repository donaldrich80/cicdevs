## Develop in Docker

### Command

```sh
docker pull donaldrich/hugo:latest
docker run -it --rm \
--hostname=hugo \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/zsh" \
--net="host" \
donaldrich/hugo:latest
```
