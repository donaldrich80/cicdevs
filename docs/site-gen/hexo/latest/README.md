## Develop in Docker

### Command

```sh
docker pull donaldrich/hexo:latest
docker run -it --rm \
--hostname=hexo \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/zsh" \
--net="host" \

```
