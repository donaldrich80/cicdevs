# donaldrich/hexo:icarus

## Description

### Purpose

Container for generating Hexo

### Features

* Hexo

* Node

* Alpine

## Develop in Docker

### Command

```sh
docker pull donaldrich/hexo:icarus
docker run -it --rm \
--hostname=hexo \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/sh" \
--net="host" \
donaldrich/hexo:icarus
```
