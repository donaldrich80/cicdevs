## Develop in Docker

### Command

```sh
docker pull donaldrich/hexo:slim
docker run -it --rm \
--hostname=hexo \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/usr/bin/zsh" \
--net="host" \
donaldrich/hexo:slim
```
