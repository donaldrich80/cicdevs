# donaldrich/hexo:butterfly

## Description

### Purpose

Container for generating Hexo

### Features

* Hexo

* Node

* Butterfly theme installed

## Develop in Docker

### Command

```sh
docker pull donaldrich/hexo:butterfly
docker run -it --rm \
--hostname=hexo \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/sh" \
--net="host" \
donaldrich/hexo:butterfly
```
