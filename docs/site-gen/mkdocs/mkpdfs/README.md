# donaldrich/mkdocs:mkpdfs

## Description

### Purpose

Container for generating a pdf from site

### Features

* mkpdfs

* mkdocs

## Develop in Docker

### Command

```sh
docker pull donaldrich/mkdocs:mkpdfs
docker run -it --rm \
--hostname=mkdocs \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/sh" \
--net="host" \
donaldrich/mkdocs:mkpdfs
```
