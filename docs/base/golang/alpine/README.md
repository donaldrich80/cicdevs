# donaldrich/base:golang.alpine

## Description

### Purpose

Alpine image for use as golang base or sandbox

### Features

- golang

- alpine

- zsh

- nano

- git

## Develop in Docker

### Command

```sh
docker pull donaldrich/base:golang.alpine
docker run -it --rm \
--hostname=base \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/zsh" \
--net="host" \
donaldrich/base:golang.alpine
```
