## Description

### Purpose

Redhat UBI image for use as golang base or sandbox

### Features

- golang

- alpine

- zsh

- nano

- git

## Develop in Docker

### Command

```sh
docker pull donaldrich/base:golang.redhat
docker run -it --rm \
--hostname=base \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/sh" \
--net="host" \
donaldrich/base:golang.redhat
```
