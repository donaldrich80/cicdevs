## Develop in Docker

### Command

```sh
docker pull donaldrich/base:alpine
docker run -it --rm \
--hostname=base \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/zsh" \
--net="host" \
donaldrich/base:alpine
```
