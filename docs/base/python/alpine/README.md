# donaldrich/base:python-alpine

## Description

### Purpose

Image for use as lightweight Python base or sandbox

### Features

- python:alpine

- zsh

- nano

- git

## Develop in Docker

### Command

```sh
docker pull donaldrich/base:python-alpine
docker run -it --rm \
--hostname=base \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="bin/zsh" \
--net="host" \
donaldrich/base:python-alpine
```
