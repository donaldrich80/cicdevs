- nano

## Develop in Docker

### Command

```sh
docker pull donaldrich/base:ubuntu
docker run -it --rm \
--hostname=base \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/zsh" \
--net="host" \
donaldrich/base:ubuntu
```
