# donaldrich/base:ubuntu

## Develop in Docker

### Command

```sh
docker pull donaldrich/base:ubuntu
docker run -it --rm \
--hostname=base \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/sh" \
--net="host" \
donaldrich/base:ubuntu
```

## Inspect Container

???+ info "Container Check"

    === "Validate Services"
