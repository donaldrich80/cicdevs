## Description

### Purpose

Image for use as lightweight Opensuse Leap base or sandbox

### Features

- Opensuse Leap

- zsh

- nano

- git

## Develop in Docker

### Command

```sh
docker pull donaldrich/base:opensuse.leap
docker run -it --rm \
--hostname=base \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/usr/bin/zsh" \
--net="host" \
donaldrich/base:opensuse.leap
```
