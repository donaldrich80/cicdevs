## Description

### Purpose

Image for use as lightweight Opensuse Python base or sandbox

### Features

- Opensuse

- zsh

- nano

- git

## Develop in Docker

### Command

```sh
docker pull donaldrich/base:opensuse.tumbleweed
docker run -it --rm \
--hostname=base \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/usr/bin/zsh" \
--net="host" \
donaldrich/base:opensuse.tumbleweed
```
