# donaldrich/base:node-alpine

## Description

### Purpose

Image for use as Node base or sandbox

### Features

* Node

* alpine

* zsh

* nano

* git

## Develop in Docker

### Command

```sh
docker pull donaldrich/base:node-alpine
docker run -it --rm \
--hostname=base \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/zsh" \
--net="host" \
donaldrich/base:node-alpine
```
