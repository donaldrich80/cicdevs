## Description

### Purpose

Image for use as Node base or sandbox

### Features

* Node

* slim

* zsh

* nano

* git

## Develop in Docker

### Command

```sh
docker pull donaldrich/base:node.slim
docker run -it --rm \
--hostname=base \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/zsh" \
--net="host" \
donaldrich/base:node.slim
```
