## Description

### Purpose

Docker Image which has no image dependencies

## Develop in Docker

### Command

```sh
docker pull donaldrich/docker:bootstrap
docker run -it --rm \
--hostname=docker \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/sh" \
--net="host" \
donaldrich/docker:bootstrap
```
