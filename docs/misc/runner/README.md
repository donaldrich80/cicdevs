## Description

### Purpose

A simple, lightweight Alpine image for performing simple CI tasks

### Features

* git

* curl

* Hashicorp Vault

* ssh

* rsync

## Develop in Docker

### Command

```sh
docker pull donaldrich/runner:latest
docker run -it --rm \
--hostname=runner \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/sh" \
--net="host" \
donaldrich/runner:latest
```
