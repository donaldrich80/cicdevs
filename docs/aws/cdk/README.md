## Description

### Purpose

Image for interacting with AWS cdk

### Features

* AWS cdk

## Develop in Docker

### Command

```sh
docker pull donaldrich/aws:cdk
docker run -it --rm \
--hostname=aws \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/usr/bin/zsh" \
--net="host" \
donaldrich/aws:cdk
```
