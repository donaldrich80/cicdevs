## Description

### Purpose

Image for interacting with AWS

### Features

- AWS command line interface

## Develop in Docker

### Command

```sh
docker pull donaldrich/aws:localstack
docker run -it --rm \
--hostname=aws \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/usr/bin/zsh" \
--net="host" \
donaldrich/aws:localstack
```
