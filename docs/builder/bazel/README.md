# donaldrich/builder:bazel

## Description

### Purpose

Builder container with bazel

### Features

- inherits all capabilities of builder:latest

- layering reduces size of extra features

## Develop in Docker

### Command

```sh
docker pull donaldrich/builder:bazel
docker run -it --rm \
--hostname=builder \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/usr/bin/zsh" \
--net="host" \
donaldrich/builder:bazel
```
