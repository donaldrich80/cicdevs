# donaldrich/builder:python

## Description

### Purpose

Builder image with Python and pip.

### Features

* Python

## Develop in Docker

### Command

```sh
docker pull donaldrich/builder:python
docker run -it --rm \
--hostname=builder \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/usr/bin/zsh" \
--net="host" \
donaldrich/builder:python
```
