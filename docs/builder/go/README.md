# donaldrich/builder:go

## Description

### Purpose

Builder container with goland

### Features

- inherits all capabilities of builder:latest

- layering reduces size of extra features

## Develop in Docker

### Command

```sh
docker pull donaldrich/builder:go
docker run -it --rm \
--hostname=builder \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/usr/bin/zsh" \
--net="host" \
donaldrich/builder:go
```
