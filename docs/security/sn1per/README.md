# donaldrich/sn1per

## Develop in Docker

### Command

```sh
docker pull donaldrich/sn1per:latest
docker run -it --rm \
--hostname=sn1per \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/sh" \
--net="host" \
donaldrich/sn1per:latest
```
