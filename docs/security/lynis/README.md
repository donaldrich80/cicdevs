# donaldrich/lynis

## Develop in Docker

### Command

```sh
docker pull donaldrich/lynis:latest
docker run -it --rm \
--hostname=lynis \
-v "$(pwd)":"/work" -w "/work" \
--entrypoint="/bin/sh" \
--net="host" \
donaldrich/lynis:latest
```
